### 安装选项

-   **VS Code 市场（推荐）：** 标准 VS Code 和 Cursor 用户的最快方法。
-   **Open VSX 注册表：** 适用于 VSCodium 等 VS Code 兼容编辑器。

### 🛠️ VS Code 市场：逐步设置

按照以下步骤启动和运行 Cline：

1.  **打开 VS Code：** 启动 VS Code 应用程序。

    > ⚠️ **注意：** 如果 VS Code 显示"运行扩展可能会..."，请点击"允许"。

2.  **打开您的 Cline 文件夹：** 在 VS Code 中，打开您在文档中创建的 Cline 文件夹。
3.  **导航到扩展：** 点击 VS Code 侧边活动栏中的扩展图标（`Ctrl + Shift + X` 或 `Cmd + Shift + X`）。
4.  **搜索 'Cline'：** 在扩展搜索栏中，输入 `Cline`。

1. **安装扩展：** 点击 Cline 扩展旁边的"安装"按钮。
2. **打开 Cline：**
    - 点击活动栏中的 Cline 图标。
    - 或者，使用命令面板（`Ctrl/Cmd + Shift + P`）并输入"Cline: 在新标签页中打开"以获得更好的视图。
3. **故障排除：** 如果您看不到 Cline 图标，请尝试重启 VS Code。

> ✅ **专业提示：** 您应该会在 VS Code 编辑器中看到 Cline 聊天窗口出现！

### 🌐 Open VSX 注册表

对于没有市场访问权限的 VS Code 兼容编辑器（如 VSCodium 和 Windsurf）：

1. 打开您的编辑器。
2. 访问扩展视图。
3. 搜索"Cline"。
4. 选择 saoudrizwan 的"Cline"并点击**安装**。
5. 如果提示，请重新加载。

### 👤 创建您的 Cline 账户

现在您已经安装了 Cline，让我们为您设置账户：

1. **登录 Cline：**
    - 点击 Cline 扩展中的**登录**按钮。
    - 您将被带到 [app.cline.bot](https://app.cline.bot) 创建您的账户。
2. **从免费积分开始：**
    - 无需信用卡！
3. **可用的 AI 模型：**
    - Anthropic Claude 3.5-Sonnet（推荐用于编程）
    - DeepSeek Chat（经济实惠的替代方案）
    - Google Gemini 2.0 Flash
    - 还有更多——全部通过您的 Cline 账户。

### 💻 您与 Cline 的第一次互动

您已经准备好开始构建了！将此提示复制并粘贴到 Cline 聊天窗口中：

```
嘿 Cline！您能帮我在我的 Cline 目录中创建一个名为"hello-world"的新项目文件夹，并制作一个简单的网页，用大号蓝色文字显示"Hello World"吗？
```

> ✅ **专业提示：** Cline 将帮助您创建项目文件夹并设置您的第一个网页！

### 🧩 使用 Cline 的技巧

-   **提问：** 如果您对某事不确定，请询问 Cline！
-   **使用截图：** Cline 可以理解图像——向他展示您正在处理的内容。
-   **复制和粘贴错误：** 在聊天中分享错误消息以获得解决方案。
-   **简单表达：** 使用您自己的话——Cline 会将它们转换为代码。
